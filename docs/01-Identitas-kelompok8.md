## Arsitektur Aplikasi

Proyek ini menggunakan konsep service-based architecture yang terdiri dari aplikasi frontend dan backend yang saling terhubung menggunakan REST API. Frontend bertugas menampilkan data dan antarmuka pengguna, sedangkan backend Laravel menangani logika bisnis, autentikasi, pengelolaan database, dan integrasi dengan third-party API.

**Aplikasi 1 — Frontend**

- Nama Aplikasi: MovieHub Frontend
- Deskripsi Singkat: Aplikasi web yang digunakan pengguna untuk mencari dan melihat informasi film.
- Berkomunikasi dengan: MovieHub Backend API

**Aplikasi 3 — Backend (Laravel)**

- Nama Aplikasi / Service: MovieHub Backend Service
- Deskripsi Singkat: Layanan backend berbasis Laravel yang menyediakan API, autentikasi pengguna, dan integrasi dengan third-party API film.
- Menyediakan layanan untuk: MovieHub Frontend
